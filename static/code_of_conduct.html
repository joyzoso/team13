Code of Conduct



We want ALL visitors to our website to have a safe, friendly, and educational experience.

This will be a place where women can feel free to ask questions of other women in the tech industry and get respectful, helpful answers.

Accordingly, users of our site and any contributions they make must comply with this Code. Your use of this site means that you accept and agree to abide by this Code of Conduct.

If you reasonably believe that any contribution to our websites made by another user contravenes this Code of Conduct and/or any of the Website Terms, please notify us using the "Report Abuse"/"Flag as Inappropriate" button.
